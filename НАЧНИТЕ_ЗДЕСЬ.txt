⚡ ПОСЛЕ ДЕПЛОЯ (ОБЯЗАТЕЛЬНО!):

1. Настройте БД в Vercel Dashboard
   → Storage → Create Database → Postgres

2. Добавьте JWT_SECRET
   → Settings → Environment Variables
   → Сгенерируйте: node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"

3. Примените миграции:
   vercel env pull .env.production.local
   cd backend && npx prisma migrate deploy && npx prisma db seed
